# LinkedIn Search Automator
A powerful tool for automating your LinkedIn job searches

## Support & Functionality
- Custom user accounts & personal pages
- Track and view history of all searches
- Smooth login/out experience
- Intuitive, responsive UI

## Security & Reliability
- Passwords stored securely with hashing + salts
- CSRF-Tokens on all forms
- IP-blocking to prevent flooding
- Single-request enforcement (no simultaneous jobs)
- Asynchronous processing courtesy of Flask
      
## Built With
- Python 3.12
- Flask
- Selenium
- SQLite3
- Pandas
